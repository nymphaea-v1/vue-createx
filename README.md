# vue-createx

Верстка по макету онлайн-курсов **Createx**.

В процессе! На данный момент я доверстываю макет и планирую добавить адаптивность/отзывчивость.

## Используемые технологии
- Vue.js 3
- TypeScript
- Препроцессор SASS (SCSS)
- webpack
- ESLint, Stylelint
